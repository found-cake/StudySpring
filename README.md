## 사용된 Lombok 어노테이션
| 이름                      | 위치            | 설명                                                          |
|-------------------------|---------------|-------------------------------------------------------------|
| RequiredArgsConstructor | class         | 기본 값이 없는 final이나 NonNull이라는 어노테이션이 없는 프로버티에 대한 생성자를 자동으로 생성 |
| Getter                  | class or 프로버티 | Getter 메소드들을 자동으로 생성                                        |
| Builder                 | class         | Builder 패턴을 구현                                              |
| NoArgsConstructor       | class         | 파라미터가 없는 생성자를 생성                                            |
